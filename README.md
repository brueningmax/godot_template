###### 🛠️ Project Setup \& Renaming

###### This template includes two scripts to help you get started quickly:

###### 

###### 1\. setup.bat6

###### &nbsp;(Full Automation)

###### Use this for a fresh project in an empty folder.

###### 

###### Automatically initializes a new Git repository.

###### Pulls the latest version of this template.

###### Prompts for your Project Name.

###### Recursively renames all references of TEMPLATE\_PROJECT\_NAME (in project.godot and other files) to your new project name.

###### Starts you with a clean Git history.

###### 

###### 2\. rename.bat

###### &nbsp;(Standalone Renaming)

###### Use this if you have already cloned or copied the template files manually.

###### 

###### Prompts for your Project Name.

###### Recursively updates all project references in the current folder and subfolders.

###### Safe to delete after use.

###### 

###### 🚀 Getting Started

###### Copy either setup.bat or rename.bat to your project folder.

###### Double-click the script and follow the prompts.

###### Your Godot project is ready
